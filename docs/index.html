wildfire-exposure-eo

Which schools, substations, water plants and fire stations in a Portuguese fire district are most exposed to wildfire? This map ranks … OSM-derived critical-infrastructure assets in the Sever do Vouga / Albergaria-a-Velha / Oliveira de Azeméis pilot AOI by relative wildfire exposure, built entirely from open data with full per-asset provenance. Built for municípios, civil-protection planners and open-data auditors: every number is reproducible from the repository.

The exposure score is a relative, AOI-normalised screening rank — not a fire probability, not a forecast, not an operational service. Public-data demonstrator.

Prithvi-EO-2.0 burn-scar inference score (de-grid p85 composite, Jun 2025–Jun 2026 Sentinel-2 window): detection of already-burned areas. As a recent-scar detector against ICNF 2023–2025 burns it reaches F1 0.64 (precision 0.81 at the 0.5 threshold). Shown with value-driven opacity — transparent below 0.25, low band lifted so the full recent-scar extent shows — so genuine scars stand out; the underlying COG is a continuous relative score, not a calibrated probability and not a fire forecast. Streamed from Cloudflare R2 (~46 MB COG; loads on toggle).

Current observed fire weather.

The operational second axis: current observed reanalysis fire weather from the CEMS Early Warning Data Store (Canadian FWI system). This is regional context over the Iberian Peninsula (mainland Portugal + Spain) on a coarse 0.25° grid (≈25 km cells), with a ~2-day lag — not a per-asset score, not a probability, not a forecast. Ocean and no-coverage cells are transparent. Read it alongside the exposure rank: the rank is the validated structural axis, this is the current weather axis. Streamed from Cloudflare R2 (loads on toggle).

Four additional Portuguese AOIs the rank was reproduced on (Pedrógão Grande, Serra da Estrela, Peneda-Gerês, Monchique), each scored independently with the same open-data pipeline and shown with the same viridis exposure-rank encoding.         Asset analyser

        A filterable, sortable table of every scored critical-infrastructure asset across the AOIs opens as a floating attribute table over the map (classic GIS style): drag it by its header, dock it to a side, collapse it, or resize it. Impact severity = exposure score × importance — a transparent relative within-AOI triage rank that floats the most-exposed important assets (e.g. high-voltage lines and substations) to the top. Not a probability. Sort by it or filter, then click a row to fly the map there; clicking an asset on the map scrolls the table to its row. Roads/highways are not in the scored set — see the note in the table.

        Scope boundaries

        • Rank, not probability. The score orders assets within this AOI; it makes no calibrated-probability or forecast claim.
        • AOI-relative. Features are percentile-ranked within the evaluated AOI; scores are not comparable across AOIs.
        • Public-data demonstrator — not an operational service.
        • Burn scars ≠ ignition prediction. The Prithvi layer detects spectral signatures of fires that already happened.
