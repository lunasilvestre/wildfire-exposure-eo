wildfire-exposure-eo

Which schools, substations, water plants and fire stations in a Portuguese fire district are most exposed to wildfire? This map ranks … OSM-derived critical-infrastructure assets in the Sever do Vouga / Albergaria-a-Velha / Oliveira de Azeméis pilot AOI by relative wildfire exposure, built entirely from open data with full per-asset provenance. Built for municípios, civil-protection planners and open-data auditors: every number is reproducible from the repository.

The exposure score is a relative, AOI-normalised screening rank — not a fire probability, not a forecast, not an operational service. Public-data demonstrator.

    Scope boundaries

    • Rank, not probability. The score orders assets within this AOI; it makes no calibrated-probability or forecast claim.
    • AOI-relative. Features are percentile-ranked within the evaluated AOI; scores are not comparable across AOIs.
    • Public-data demonstrator — not an operational service.
    • Burn scars ≠ ignition prediction. The Prithvi layer detects spectral signatures of fires that already happened.
