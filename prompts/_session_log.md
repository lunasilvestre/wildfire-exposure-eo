# Claude Code session log

Append a brief entry per session. Format:

```
## YYYY-MM-DD — <prompt-file>
- Commits: <sha1>, <sha2>
- Gates passed: ruff ✓ format ✓ pyright ✓ pytest ✓ stac-validator (n/a) smoke ✓
- Notes: <one paragraph; constraints hit, deviations from prompt, follow-ups>
```

Keep entries terse. The git log carries the detail; this log carries the narrative.
